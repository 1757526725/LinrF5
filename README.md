# LinrF5
一款自动刷新浏览器的小工具，它通过监听用户的按键，如果 Ctrl+S 被按下，则自动为您刷新浏览器，无需您再次按下 F5 键，提高开发效率。
如果您之前有使用过 SuperApp，您应该对这个功能很熟悉，现在我把这个功能单独提起出来给大家使用，希望您喜欢 ^^*
